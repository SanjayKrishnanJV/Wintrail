## First-Time Setup

### Prerequisites
- **macOS** with Xcode 15+ (for iOS)
- **Android Studio** with SDK 33+ (for Android)
- Node 18+

### Install dependencies
```bash
npm install
```

### Regenerate icons/splash after changing resources/icon.svg or resources/splash.svg
```bash
npm run icons
```

## iOS — App Store Submission

### 1. Open in Xcode
```bash
npm run cap:ios   # builds + opens Xcode
```

### 2. Configure signing
- In Xcode → **App target → Signing & Capabilities**
- Set your **Team** (Apple Developer account)
- Bundle ID is already set to `com.wintrail.app`

### 3. Set version
- `MARKETING_VERSION` = e.g. `1.0.0`
- `CURRENT_PROJECT_VERSION` = `1`

### 4. Archive & upload
- In Xcode menu: **Product → Archive**
- Click **Distribute App → App Store Connect → Upload**

### 5. App Store Connect
- Log in to [appstoreconnect.apple.com](https://appstoreconnect.apple.com)
- Create a new app with bundle ID `com.wintrail.app`
- Fill in metadata, screenshots, description
- Select the uploaded build → Submit for Review

### Required App Store metadata
| Field | Value |
|---|---|
| Name | Wintrail |
| Subtitle | Your AI Learning Architect |
| Category | Education |
| Content Rating | 4+ |

| Support URL | https://learnflow-ai-two.vercel.app/support.html |

## Android — Google Play Submission

### 1. Open in Android Studio
```bash
npm run cap:android   # builds + opens Android Studio
```

### 2. Set version in android/app/build.gradle
```gradle
android {
  defaultConfig {
    versionCode 1
    versionName "1.0.0"
  }
}
```

### 3. Create a signed APK / AAB
- **Build → Generate Signed Bundle / APK**
- Choose **Android App Bundle (.aab)** (required for Play Store)
- Create or select a keystore — **keep your keystore file safe!**

### 4. Google Play Console
- Log in to [play.google.com/console](https://play.google.com/console)
- Create a new app with package name `com.wintrail.app`
- Upload the `.aab` file to **Production** track
- Fill in store listing, screenshots, content rating
- Submit for review

### Required Play Store metadata
| Field | Value |
|---|---|
| App name | Wintrail |
| Short description | AI-powered personalised learning roadmaps |
| Category | Education |
| Content rating | Everyone |
| Privacy Policy URL | https://learnflow-ai-two.vercel.app/privacy.html |

## Environment

The mobile build uses `.env.mobile` which sets:
```
VITE_API_BASE=https://learnflow-ai-two.vercel.app
```

This means API calls (`/api/mentor`, `/api/roadmap`) go to the Vercel deployment.  
Update this URL if the Vercel project is renamed or moved.

## File Structure

```
ios/          → Xcode project (generated by Capacitor)
android/      → Android Studio project (generated by Capacitor)
resources/
  icon.svg          → Source icon (edit this)
  splash.svg        → Source splash screen (edit this)
  icon.png          → 1024×1024 (auto-generated)
  splash.png        → 2732×2732 (auto-generated)
scripts/
  generate-icons.mjs  → Generates all icon sizes from SVG sources
capacitor.config.json → Capacitor configuration
.env.mobile           → Env vars for the mobile bundle build
```
